modal window for landing page

Модальное окно

Скрипт модального окна написан на js. Само окно скрыто с помощью display='none' в таблице стилей. При клике на кнопку окно открывается. При нажатии на темный фон или кнопку "закрыть" закрывается. Для всех перечисленных событий кликов соответственно написаны обработчики событий в js-файле.

Особенность окна

"Фишка" скрипта в том, что он добавляет скрытое инпут-поле к форме, которая появляется в модальном окне. В landing page часто встречается перечень каких-то продуктов, которые может заказать пользователь. И для заказа отправляется форма обратной связи. Поэтому удобно, когда сразу на почту приходит название того, товара, который пользователь хотел заказать.

Как работает

Чтобы модальное окно работало, нужно кнопке прописать класс "popup-link" и атрибут "data-href" со ссылкой на id окна.
А для того, чтобы срипт подставлял скрытое поле в форму в модальном окне, кнопке прописываются атрибуты data-formId (id формы), data-inputName (name скрытого поля), data-value (значение скрытого поля)

css3 tabs

css3 вкладки

Вкладки сделаны на чистом css. Для переключения используются скрытые радио-баттоны. Кнопки чуть выше это теги label, связанные с этими радио-баттонами. Блоки с текстом сделаны абсолютным позиционированием и с помощью opacity.

css3 слайдер

Слайдер сделан по тому же принципу, что и вкладки. С помощью радио-баттонов происходит переключение слайдов, а с помощью transition происходит небольшой плавный переход от одной фотки к другой.

flex responsive сетка

Сетка страницы сочетает в себе две технологии, display: flex и responsive web design. Выравнивание блоков по центру при изменении разрешения экрана очень удобно организовать с помощью flexbox. А для создания адаптивных макетов на основе 12-колоночной сетки используются css-классы с определенной процентной шириной.
